# Pomfret Prompt Generator

A guided web wizard that generates high-quality, LLM-and-model-tuned study prompts for Pomfret School students.

## What it does

Students pick their LLM, their class (from the Pomfret curriculum), their study mode (cram, multi-day plan, practice questions, concept clarification, or essay prep), the assessment they're prepping for, and how confident they feel. The system produces a prompt — formatted for their specific model — that they paste into ChatGPT / Claude / Gemini for a real study session.

The prompt follows a 7-section framework (Role, About Me, Material, Goal, Interaction Style, Output Spec, Self-Check). All 7 sections are generated by Claude Opus 4.7 from a single API call — Opus reads the student's wizard inputs and the Pomfret course catalog, then writes a custom prompt tuned for the student's chosen LLM (XML for Claude, markdown for GPT, numbered steps for Gemini). If the Opus call fails or the daily budget is exhausted, the system falls back to deterministic templates.

## Tech

- Next.js 14 (App Router) + TypeScript strict
- Tailwind + shadcn/ui
- Zod validation
- `@anthropic-ai/sdk` with prompt caching
- Vercel KV (Upstash Redis) for rate limiting + feedback aggregation
- Vitest + Playwright

## Local setup

```bash
cp .env.local.example .env.local
# edit .env.local with real keys (ANTHROPIC_API_KEY required; KV optional for local)

npm install
npm run dev
```

Visit http://localhost:3100. (Port 3100 is the project default — port 3000 is reserved for the Pomfret DEI site.)

## Tests

```bash
npm test            # unit + integration (Vitest)
npm run test:e2e    # Playwright critical-path E2E
```

## Refresh course catalog

When the Pomfret curriculum guide changes (typically yearly):

```bash
# Update the SOURCE path inside scripts/build-courses.ts if needed
npm run build:courses
```

This rebuilds `data/courses.json`.

## Deploy

Push to Vercel. Set environment variables there (`ANTHROPIC_API_KEY`, `KV_REST_API_URL`, `KV_REST_API_TOKEN`, plus optional knobs `RATE_LIMIT_PER_IP_PER_DAY`, `DAILY_BUDGET_CEILING_USD`).

## Privacy

Pasted material is never persisted (no DB, no logs — errors redact body). Prompt history is stored only in the user's browser localStorage. Feedback is anonymous, stored against a SHA-256 of the prompt.
